A unique ID generator. Default is {@link UUIDGenerator}. Here are some examples of the IDs it makes: uuri:0161811f-5da6-4c6e-9808-a2fab97114cf;type=metadata and uuri:6f2721ea-d732-43d7-bf49-6b16ecb88557;type=request. To use another ID Generator, set the System Property org.archive.uid.GeneratorFactory.generator to point at an alternate implementation of {@link Generator}.
