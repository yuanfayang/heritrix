Provides specializations on apache jakarta commons httpclient.

HttpRecorderProtocolSocketFactory

Returns sockets that have their streams wrapped by {@link org.archive.util.HttpRecorder}.

HttpRecorderSSLProtocolSocketFactory

Returns sockets that have their streams wrapped by {@link org.archive.util.HttpRecorder} AND they have configurable TrustManager. Built upon HttpRecorderProtocolSocketFactory.
