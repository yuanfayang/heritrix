Provides heritrix specializations on the apache jakarta commons httpclient.

Trust Stores

When our httpclient trust setting is configured to anything but OPEN, we use the truststore specified by the javax.net.ssl.trustStore property set in the heritrix.properties file IF there is no javax.net.ssl.trustStore system property defined on the command line. If so, we use whatever the commandline points at instead.
